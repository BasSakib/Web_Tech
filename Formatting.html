HTML is the standard language used to create web pages. It provides the structure of a website and allows developers to organize content using different tags.

In this lab, we are learning the basic structure of HTML documents and how to use various formatting tags like highlight, small text, and line breaks.
This helps in understanding how web content is displayed in browsers.

Practice is very important for learning HTML. By completing this task, students will gain a strong foundation in web development.
